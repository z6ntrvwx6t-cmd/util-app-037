# MCOL Designer

Pricing calculator for Music City Outdoor Lighting.

## Setup

1. Copy `.env.example` to `.env` and fill in your Supabase credentials
2. `npm install`
3. `npm run dev`

## Features

- 4-zone lighting pricing calculator
- 3 tiers: Premium-RGB, Premium, Essential
- Audio system pricing
- Protection plan calculator
- 12-month financing calculations

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (auth + database)
